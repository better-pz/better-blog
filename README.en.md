前端: 
Web前端是网络平台上，展示、传递信息给用户，或可以跟用户进行交流的界面,前端开发 通过各种技术手段，完成页面的开发和重构以及完成与服务端的接口对接联调，将信息美观的在各种浏览器/软件上展示给用户/浏览者 并且实现用户与的交流即交互功能。
